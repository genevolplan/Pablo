1 Obtención de secuencias crudas aleatorias de Moringa oleifera

Para la obtención de secuencias crudas aleatorias de esta especie, se hizo uso de la accesión SRR34565303, obtenida en el repositorio SRA. Se seleccionó aquella con un mayor longitud de bases y número de spots.

Como se indica en la imagen, se empleó la plataforma PacBio Sequel II.

Para la extracción de un archivo fasta con las lecturas a partir de esta accesión, se empleó la herramienta SRA toolkit(), empléandose el siguiente comando:

# prefetch SRR34565303 -O /home/kubuntu/apps/sratoolkit/filedump/SRR34565303
# fasterq-dump /home/kubuntu/apps/sratoolkit/filedump/SRR34565303

3 Ensamblaje de genoma mitocondrial mediante secuencias crudas aleatorias

Partiendo del archivo en formato fasta obtenido mediante SRA toolkits, se realizó el ensamblaje del genoma mitocondrial empleando la herramienta oatk().

Para ello, se empleó el siguiente comando

# oatk -k 1001 -c 30 -t 16 --nhmmscan /home/kubuntu/anaconda3/bin/nhmmscan -m /home/kubuntu/analysis/synology/Utilities/DBs/OatkDB/v20230210/angiosperm_mito.fam -p /home/kubuntu/analysis/synology/Utilities/DBs/OatkDB/v20230210/angiosperm_pltd.fam -o MORINGASTN /home/kubuntu/analysis/synology/M_oleifera_SRR34565303/SRR34565303.fastq.gz

Donde:

-k representa el tamaño de cada syncmer. Se conservó el valor por defecto.

-c representa el threshold de cobertura para cada syncmer. Se conservó el valor por defecto.

-t representa el número de hilos destinados a realizar el ensamblaje. Dado que el ordenador empleado presentaba 20 hilos en total, se seleccionó una cantidad conservadora de 16.

-m indica donde debe incluirse el archivo .fam correspondiente a la base de datos de genomas mitocondriales de angiospermas.

-p indica donde debe incluirse el archivo -fam correspondiente a la base de datos de genomas plastídicos de angiospermas.

-o especifica el prefijo de los archivos de salida.

4 Representación visual del ensamblaje del genoma plastídico

A partir del archivo .gfa generado tras el ensamblaje, se obtuvo una representación gráfica del mismo mediante el programa Bandage().

En este tipo de gráfica se representa información de 3 parámetros relevantes:

  • La longitud de una región de ensamblaje en pb.
  • El carácter circularizado o lineal de la misma, representado gráficamente.
  • La cobertura media de esta región.

Con esta información, puede analizarse el resultado obtenido en la figura. El objeto obtenido en la parte inferior presenta la morfología usual esperada para un genoma plastídico, teniendo en cuenta que este presenta una región larga de copia única (LSC) representada por el bucle de mayor tamaño, una región corta de copia única (SSC) por el de menor, y regiones repetidas inversas (IRs) correspondientes a la zona central del gráfico, de 26535 pb.

Se aprecia, no obstante, una región larga y aislada, de más de 160.000 pb y de carácter no circularizado. Esto supone una desviación del resultado esperado, por lo que para arrojar una interpretación se realizó un estudio de homología de esta región con respecto al genoma plastídico de referencia para Moringa oleifera.

5 Estudio de identidad entre ensamblaje del genoma plastídico de Moringa oleifera y genoma de plastídico de referencia

En primer lugar, se realizó el estudio para el contig ctg000001c, asociado a la región circularizada del ensamblaje.

Tras esto, se realizó el mismo análisis para el contig ctg000002l, asociado a la región anómala no circularizada detectada.

Esta región, de un tamaño muy similar a la primera y cercano al esperable para el genoma plastídico podría ser fruto de un artefacto derivado del propio proceso de ensamblaje mediante Otak.
